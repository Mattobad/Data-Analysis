    Answering Business Questions using SQL

    1. Introduction and Schema Diagram¶

    In this project, we're going to practice using our SQL skills to answer business questions. For this, we'll use Chinook database, a media related data created using real data from an iTunes Library. You can find more about the database here.The Chinook database is provided as a SQLite database file called chinook.db.

    Summary of the Results:

    Here, we used different SQL queries to answer our questions related to Chinook database: media related data from iTunes Library.

    We concluded that USA has the highest number of customer of 13 and total transaction of 131 resulting in total sales of 1040.49 dollars. But Czech Rep has the highest average sales per customer with 136.62 dollars with only 2 customer and 30 total transaction. So, eventhough USA has highest total sales due to higher number of customers as well as higher number transaction, it lies fourth from last in average sales per customer.

WITH

    (

    Second, we're putting the countries with only 1 customer to Other category. Then, we're creating custom sort with all the countries sorted in descending order except putting Other category at bottom.

    In [15]:
    %%sql

WITH 
customer_info AS
    (
        SELECT 
            CASE
                WHEN COUNT(DISTINCT c.customer_id)  == 1 THEN 'Other'
                ELSE c.country
                END
                AS derived_country,       
            COUNT(DISTINCT c.customer_id) number_of_customers,
            COUNT(i.invoice_id) transaction_count,
            ROUND(SUM(i.total),2) total_sales
        FROM customer c
        INNER JOIN invoice i ON i.customer_id = c.customer_id 
        GROUP BY c.country
        ORDER BY 4 DESC
    
    ),
    
customer_by_derived_country AS
    (
        SELECT 
            derived_country country,
            SUM(number_of_customers) number_of_customers,
            SUM(transaction_count) total_transaction,
            total_sales
        FROM customer_info
        GROUP BY 1
    )
    

SELECT 
    country,
    number_of_customers total_number_of_customers,
    total_transaction,
    total_sales,
    ROUND((total_sales / number_of_customers),2) avg_sales_per_customer,
    ROUND((total_sales/ total_transaction),2) avg_order_value
FROM (  
        SELECT 
            cd.*,
            CASE
                WHEN cd.country = 'Other' THEN 1
                ELSE 0
            END AS sort
        FROM customer_by_derived_country cd
        ORDER BY cd.number_of_customers DESC
    )
ORDER BY sort ASC;
